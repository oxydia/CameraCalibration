ce programme utiliser la bibliothèque openKraken.

compilation : > make
utilisation : > ./pointListExporter [path]/monImage.jpg

click gauche : sélectione un point 
flèches : déplace ce point pour l'ajuster correctement
molette : zoom
click droit : déplacer l'image en cas de zoom
echap : sort du programme et sauvegarde la liste de points dans le fichier [path]/monImage.jpg.list
